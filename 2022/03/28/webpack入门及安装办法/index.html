webpackеҝ«йҖҹе…Ҙй—Ё

д»Җд№ҲжҳҜwebpack

е®ҳж–№зҪ‘еқҖпјҡwebpackjs.com
иҮӘеҠЁеҢ–жү“еҢ…е·Ҙе…·пјҢдјҳеҢ–жҲ‘д»¬зҡ„йЎ№зӣ®пјҢжңҚеҠЎиҜ·жұӮжӣҙеҝ«гҖӮ
webpackжҳҜзҺ°д»ЈJavaScriptеә”з”ЁзЁӢеәҸзҡ„йқҷжҖҒжү“еҢ…е·Ҙе…·пјҢеҪ“webpackеӨ„зҗҶеә”з”ЁзЁӢеәҸж—¶пјҢе®ғдјҡеңЁеҶ…йғЁжһ„е»әдёҖдёӘдҫқиө–еӣҫпјҢжӯӨдҫқиө–еӣҫдјҡжҳ е°„йЎ№зӣ®жүҖйңҖзҡ„жҜҸдёӘжЁЎеқ—пјҢе№¶з”ҹжҲҗдёҖдёӘжҲ–еӨҡдёӘbundleеҢ…гҖӮ

дёәе•ҘиҰҒдҪҝз”Ёwebpack

д»Јз ҒиҪ¬жҚў:TypeScriptзј–иҜ‘жҲҗJavaScriptпјҢLESS/SCSSзј–иҜ‘CSSпјҢES6/7зј–иҜ‘жҲҗES5пјҢиҷҡжӢҹDOMзј–иҜ‘жҲҗзңҹе®һDOMзӯүзӯү
ж–Үд»¶дјҳеҢ–:еҺӢзј©jsпјҢcssпјҢHTMLд»Јз ҒпјҢеҺӢзј©еҗҲе№¶еӣҫзүҮпјҢеӣҫзүҮbase64зӯү
д»Јз ҒеҲҶеүІ:жҸҗеҸ–еӨҡдёӘйЎөйқўзҡ„е…¬е…ұд»Јз ҒгҖҒжҸҗеҸ–йҰ–еұҸдёҚйңҖиҰҒжү§иЎҢйғЁеҲҶзҡ„д»Јз Ғ
жЁЎеқ—еҗҲе№¶:еңЁйҮҮз”ЁжЁЎеқ—еҢ–зҡ„йЎ№зӣ®йҮҢдјҡжңүеҫҲеӨҡдёӘжЁЎеқ—пјҢжЁЎеқ—дёӢжңүеҫҲеӨҡж–Үд»¶пјҢйңҖиҰҒжһ„е»әеҠҹиғҪжҠҠеҗ„дёӘжЁЎеқ—ж–Үд»¶еҗҲе№¶жҲҗдёҖдёӘж–Үд»¶
иҮӘеҠЁеҲ·ж–°:зӣ‘еҗ¬жң¬ең°жәҗд»Јз Ғзҡ„еҸҳеҢ–,иҮӘеҠЁйҮҚж–°жһ„е»әпјҢеҲ·ж–°жөҸи§ҲеҷЁ
д»Јз Ғж ЎйӘҢ: Eslintд»Ј з Ғи§„иҢғж ЎйӘҢе’ҢжЈҖжөӢпјҢеҚ•е…ғжөӢиҜ•
иҮӘеҠЁеҸ‘еёғ:иҮӘеҠЁжһ„е»әеҮәзәҝдёҠеҸ‘еёғзүҲжң¬пјҢдј иҫ“з»ҷеҸ‘еёғзі»з»ҹ
вҖҰвҖҰвҖҰвҖҰвҖҰвҖҰвҖҰвҖҰвҖҰ

npmпјҡе®үиЈ…webpackеүҚйңҖиҰҒдәҶи§Јзҡ„

npm = node package manager
NPMзҡ„е…Ёз§°жҳҜNode Package ManagerпјҢжҳҜдёҖдёӘNodeJSеҢ…з®ЎзҗҶе’ҢеҲҶеҸ‘е·Ҙе…·пјҢе·Із»ҸжҲҗдёәдәҶйқһе®ҳж–№зҡ„еҸ‘еёғNodeжЁЎеқ—пјҲеҢ…пјүзҡ„ж ҮеҮҶгҖӮ
еёёз”Ёзҡ„npmиҜӯжі•пјҡ

npm install xxx        //жҠҠxxxе®үиЈ…еңЁеҪ“еүҚзӣ®еҪ•
npm i xxx -g        //жҠҠxxxе®үиЈ…еңЁе…ЁеұҖ global
npm i xxx@xx.xx        //е®үиЈ…еҲ¶е®ҡзүҲжң¬еҸ·зҡ„xxx    egпјҡnpm i jquery@1.11.3
npm uninstall xxx    /-g        //еҚёиҪҪxxxеҢ…

е®үиЈ…еңЁе…ЁеұҖзҡ„дҪңз”Ёпјҡ

  1. жүҖжңүзҡ„йЎ№зӣ®йғҪиғҪз”Ё
  2. еҸҜд»ҘдҪҝз”Ёе‘Ҫд»ӨпјҲеңЁwinдёӢпјү
  3. е®үиЈ…еңЁе…ЁеұҖжЁЎеқ—дёҚиғҪеҹәдәҺCommonJsзӯүжЁЎеқ—и§„иҢғеңЁд»Јз ҒдёӯеҜје…ҘжЁЎеқ—

е®үиЈ…webpack

е®үиЈ…package.json

еңЁж №зӣ®еҪ•дёӢеҸій”®пјҢеңЁйӣҶжҲҗз»Ҳз«Ҝдёӯжү“ејҖпјҢйҰ–е…ҲжҲ‘д»¬е…Ҳе®үиЈ…дёҖдёӘpackage.jsonж–Үд»¶гҖӮеңЁз»Ҳз«Ҝдёӯиҫ“е…Ҙ

npm init -y        //еңЁеҪ“еүҚзӣ®еҪ•дёӢз”ҹжҲҗдёҖдёӘpackage.jsonж–Үд»¶пјҢжіЁж„ҸеҪ“еүҚйЎ№зӣ®зӣ®еҪ•дёӯдёҚиғҪеҢ…жӢ¬дёӯж–ҮжҲ–зү№ж®Ҡеӯ—з¬Ұ

иҰҒжіЁж„ҸnpmжӯЈеҰӮдёҠж–ҮжүҖиҜҙпјҢиҰҒеңЁnodeзҺҜеўғдёӢжүҚиғҪиҝҗиЎҢгҖӮиҰҒжҸҗеүҚе®үиЈ…еҘҪnodeпјҲе®үиЈ…ж•ҷзЁӢеҫҲеӨҡпјҢиҮӘиЎҢе®үиЈ…пјүгҖӮ
жҜҸдёӘйЎ№зӣ®зҡ„ж №зӣ®еҪ•дёӢйқўпјҢдёҖиҲ¬йғҪжңүдёҖдёӘpackage.jsonж–Үд»¶пјҢе®ҡд№үдәҶиҝҷдёӘйЎ№зӣ®жүҖйңҖиҰҒзҡ„еҗ„з§ҚжЁЎеқ—пјҢд»ҘеҸҠйЎ№зӣ®зҡ„й…ҚзҪ®дҝЎжҒҜпјҲжҜ”еҰӮеҗҚз§°гҖҒзүҲжң¬гҖҒи®ёеҸҜиҜҒзӯүе…ғж•°жҚ®пјүгҖӮnpm installе‘Ҫд»Өж №жҚ®иҝҷдёӘй…ҚзҪ®ж–Үд»¶пјҢиҮӘеҠЁдёӢиҪҪжүҖйңҖзҡ„жЁЎеқ—пјҢд№ҹе°ұжҳҜй…ҚзҪ®йЎ№зӣ®жүҖйңҖзҡ„иҝҗиЎҢе’ҢејҖеҸ‘зҺҜеўғгҖӮ

е®үиЈ…webpack

npm i webpack webpack-cli --save-dev    //--save-dev е®үиЈ…еңЁејҖеҸ‘зҺҜеўғпјҢдёҚеҠ зҡ„иҜқй»ҳи®Өе®үиЈ…еңЁз”ҹдә§зҺҜеўғ

е…ЁеұҖе®үиЈ…еҸӘиҰҒз”өи„‘дёҚеҸ‘з”ҹй—®йўҳпјҢе°ұж— йңҖеӨҡж¬Ўе®үиЈ…пјҢдҪҶжҳҜе®ҳж–№дёҚжҺЁиҚҗе…ЁеұҖе®үиЈ… webpackгҖӮиҝҷдјҡе°ҶдҪ йЎ№зӣ®дёӯзҡ„ webpack й”Ғе®ҡеҲ°жҢҮе®ҡзүҲжң¬пјҢе№¶дё”еңЁдҪҝз”ЁдёҚеҗҢзҡ„ webpack зүҲжң¬зҡ„йЎ№зӣ®дёӯпјҢеҸҜиғҪдјҡеҜјиҮҙжһ„е»әеӨұиҙҘгҖӮ

жңүдәӣж—¶еҖҷnpmе®үиЈ…дјҡйқһеёёйқһеёёж…ўпјҢиҝҷж—¶жҲ‘д»¬еҸҜд»Ҙе…Ҳе®үиЈ…yarnпјҢеҲ©з”ЁyarnеҶҚе®үиЈ…webpackдјҡеҝ«еҫҲеӨҡ

npm i yarn -g        //yarnе®үиЈ…
yarn add webpack webpack-cli -D

ж·»еҠ webpack.config.jsж–Үд»¶

еңЁж №зӣ®еҪ•дёӢеҲӣе»әж·»еҠ webpack.config.jsж–Үд»¶гҖӮе№¶ж·»еҠ еҰӮдёӢд»Јз Ғпјҡ

const path = require ('path');

module.exports = {
    //и®ҫзҪ®зј–иҜ‘зҡ„е…ҘеҸЈж–Үд»¶пјҲзңҹе®һйЎ№зӣ®дёӯдёҖиҲ¬ејҖеҸ‘зҡ„д»Јз ҒйғҪдјҡж”ҫеңЁsrcзӣ®еҪ•дёӯпјү
    entry:'./src/main.js',      
    //и®ҫзҪ®зј–иҜ‘зҡ„еҮәеҸЈж–Үд»¶
    output:{
        //зј–иҜ‘еҗҺж–Үд»¶еҗҚ
        filename:'bundle.[hash:5].js',        //hashеңЁеҶ…е®№жӣҙж–°еҗҺдјҡиҮӘеҠЁз”ҹжҲҗдёҖдёІе“ҲеёҢеҖјпјҢжқҘйҒҝе…ҚжөҸи§ҲеҷЁзҡ„ејәзј“еӯҳпјҢеҸҠж—¶жӣҙж–°ж–Үд»¶
                                                //жөҸи§ҲеҷЁзҡ„ејәзј“еӯҳпјҡеҪ“иҝҗиЎҢзҡ„ж–Үд»¶еҗҚзӣёеҗҢж—¶пјҢдјҡй»ҳи®ӨеҠ еңЁзј“еӯҳйҮҢзҡ„ж–Үд»¶
        //иҫ“еҮәзҡ„зӣ®еҪ•пјҲз»қеҜ№и·Ҝеҫ„пјү
        path:path.resolve(__dirname, 'build')
            }
             mode:"development"    //ејҖеҸ‘жЁЎејҸпјҢжІЎжңүеҜ№jsзӯүж–Үд»¶еҺӢзј©пјҢй»ҳи®Өз”ҹжҲҗзҡ„жҳҜеҺӢзј©ж–Үд»¶
    }

й…ҚзҪ®package.json

жү“ејҖpackage.jsonпјҢе°ҶscrtptsдёӢд»Јз ҒжҚўжҲҗдёӢйқўж ·ејҸ

"scripts": {
    "serve": "webpack",
  },

й…ҚзҪ®е®ҢжҲҗеҗҺпјҢжҲ‘д»¬еңЁз»Ҳз«Ҝиҫ“е…Ҙзҡ„еҰӮдёӢе‘Ҫд»ӨпјҢеҸҜд»ҘзӣҙжҺҘжү“еҢ…гҖӮ

npm run start

дјҳз§Җзӣёе…іеҚҡе®ў

иҮӘе·ұй…Қе®Ңwebpackд№ӢеҗҺе°ұжғіеҶҷзҜҮеҚҡе®ўи®°еҪ•дёҖдёӢпјҢж–№дҫҝиҮӘе·ұжҲ–еҲқеӯҰиҖ…жҹҘиҜўгҖӮеңЁеҮҶеӨҮеҚҡе®ўзҡ„иҝҮзЁӢдёӯзңӢдәҶи®ёеӨҡзӣёе…іеҚҡе®ўпјҢжңүдёҖдәӣи®©жҲ‘еҸ№дёәи§ӮжӯўпјҢдёҖеәҰеҫҲзҫһж„§иҮӘе·ұзҡ„дҪңе“Ғе“Ҳе“Ҳе“ҲпјҢеӨ§е®¶еҰӮжһңзңӢжҲ‘зҡ„дёҚеӨӘжҳҺзҷҪеңЁиҝҷйҮҢжҺЁиҚҗдёӨдёӘпјҡ
жё…жҷ°зҡ„webpackеҸҠжҸ’д»¶е®үиЈ…иҝҮзЁӢпјҡwebpackе®үиЈ…еҸҠжү“еҢ…иҜҰз»ҶиҝҮзЁӢ
webpackе®үиЈ…иҝҮзЁӢдёӯеҸҜиғҪйҒҮеҲ°зҡ„й—®йўҳеҸҠи§ЈеҶіж–№жі•пјҡwebpackе®үиЈ…еёёи§Ғй—®йўҳеҸҠеҹәжң¬жҸ’д»¶дҪҝз”Ё

жң¬дҪңе“ҒйҮҮз”Ё зҹҘиҜҶе…ұдә«зҪІеҗҚ-йқһе•ҶдёҡжҖ§дҪҝз”Ё-зҰҒжӯўжј”з»Һ 4.0 еӣҪйҷ…и®ёеҸҜеҚҸи®® (CC BY-NC-ND 4.0) иҝӣиЎҢи®ёеҸҜгҖӮ
